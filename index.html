Inbox Management

The Unified Organiser Inbox

Manage all communications from a single dashboard. Private replies automatically thread under your original message, organised by event or date.

  • Threaded replies keep every conversation in context
  • Team and CLub filtering for instant navigation
  • Read receipts — know exactly who's informed
Smart Integration

BROE2 & CSV Integration

Don't waste time on manual data entry. Connect directly to your existing event infrastructure in seconds.

  • Direct BROE2 Sync — fetch entries, and affiliations automatically via API
  • Flexible CSV uploads with row-by-row validation for non-BROE2 events
  • Duplicate detection — coaches are never spammed across multiple clubs or events
Sport-Specific Data Model

Built for Sport's Real Complexity

We understand competitive sport isn't simple. Coaches across multiple clubs, officials spanning different events — our data model handles every real-world relationship.

  • Multi-club affiliation — coaches are always reachable regardless of which club they're representing today
  • Private broadcast replies — no "Reply All" noise for everyone else
  • Read receipts — peace of mind for critical safety communications
  • Audit logs — every import logged for instant review
Guaranteed Delivery

Nobody Misses a Message.

Coaches receive instant push notifications for every update. Direct to your handheld device The system instantly notifies you so you never miss any updates, safety briefing or schedule change.
